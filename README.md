# 귀향: 폐교의 불빛 3D 1인칭 버전

기존 2D 버전을 지우고 이 파일 4개를 저장소 맨 위에 넣으면 됩니다.

## 파일
- index.html
- style.css
- game.js
- README.md

## 조작
### 패드/태블릿
- 왼쪽 조이스틱: 이동
- 오른쪽 화면 드래그: 시야 회전
- RUN 버튼: 달리기
- E 버튼: 상호작용

### PC
- WASD / 방향키: 이동
- 마우스: 시야 회전
- Shift: 달리기
- E: 상호작용

## 구현된 기능
- 3D 1인칭 시점
- 폐교 복도와 교실
- 손전등이 보는 방향으로 비춤
- 주변이 조금 보이는 어두운 분위기
- 퓨즈 5개 수집
- 전기실 전선 연결
- 차단기
- 교장실 전원
- 정전 후 처녀귀신 등장
- 부적 5장 수집
- 귀신 봉인
- 게임오버 / 클리어 화면
