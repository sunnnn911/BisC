# 귀향: 폐교의 불빛

웹으로 실행되는 2D 공포 게임 데모입니다.

## 플레이 방법

- 이동: `W A S D` 또는 방향키
- 상호작용: `E`
- 목표는 왼쪽 위에 표시됩니다.

## 게임 흐름

1. 폐교 안으로 들어가기
2. 퓨즈 5개 찾기
3. 전기실에서 전선 연결하기
4. 차단기 올리기
5. 교장실에서 전원 켜기
6. 정전 후 처녀귀신 등장
7. 부적 5장 찾기
8. 처녀귀신 봉인하기

## GitHub Pages 배포

1. 이 파일들을 저장소 최상단에 업로드합니다.
2. GitHub 저장소에서 `Settings`로 들어갑니다.
3. 왼쪽 메뉴에서 `Pages`를 누릅니다.
4. `Branch`를 `main`, 폴더를 `/root`로 선택합니다.
5. 저장하면 잠시 후 웹사이트 링크가 생성됩니다.
